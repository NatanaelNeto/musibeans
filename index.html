Musibeans

1

Conecte seu dispositivo MIDI

Conecte seu dispositivo MIDI via USB para poder jogar.

2

Identifique as notas de Dó e Si.

Nós usamos as notas centrais do teclado para controlar o jogo. Apenas as sete notas básicas. Por gentileza, aperte o Dó e o Si central do seu teclado.

3

Vamos jogar!

Cada nota é equivalente a uma cor. Nós iremos gerar uma sequência de 4 notas, e você deverá descobrí-la. Para cara nota certa no momento certo, receberá uma tecla preta. Para cada nota certa, mas no tempo diferente, receberá uma nota branca.
