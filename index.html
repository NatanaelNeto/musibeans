Como funciona

Uma sequência é gerada aleatoriamente pelo sistema.

Você deverá usar as notas do seu teclado para inserir uma combinação. Cada note é equivalente a uma cor. Dó, Ré, Mi, Fá, Sol, Lá e Si são as notas usadas e suas respectivas cores.

Após inserir uma sequência, aperte qualquer nota para confirmar.

Caso haja uma NOTA CERTA na POSIÇÃO CERTA, você recebe uma tecla preta.

Caso haja uma NOTA CERTA na POSIÇÃO ERRADA, você recebe uma tecla branca.

Se, após 10 tentativas, você não conseguir descobrir a sequência correta, você perde, e um novo jogo é começado.
