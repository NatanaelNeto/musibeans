Como jogar

Ao começar o jogo, uma sequência de notas é gerada pelo sistema.

Você possui 10 tentativas para tentar descobrir qual a sequência correta.

Com seu controlador MIDI, toque 4 das 7 notas musicais, sem repetir a mesma nota.

Notas musicais e suas respectivas cores.

Após você inserir as notas, aperte qualquer tecla para confirmar OU clique no botão abaixo para limpar a linha.

Para cada nota CORRETA na posição ERRADA, você receberá uma tecla branca. Para cada nota CORRETA na posição CORRETA, você receberá uma tecla preta.
